### 오늘 한 일

- 노마드코더 트위터 클론코딩 실습
  - FileReader API 이용하여 썸네일 등록
  - Storage 생성
  - 파일 업로드

```jsx
const onFileChange = (event) => {
    const {
      target: { files },
    } = event;
    const theFiles = files[0];
    const reader = new FileReader();
    reader.onloadend = (finishedEvent) => {
      const {
        currentTarget: { result },
      } = finishedEvent;
      setAttachment(result);
    };
    reader.readAsDataURL(theFiles);
  };
```
